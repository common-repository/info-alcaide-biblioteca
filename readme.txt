Plugin gestión de biblioteca, para prestamos en pequeños organismos

== Description ==

Plugin para una biblioteca sencilla, para mostrar libros disponibles o bajo prestamo.
Ideal para Colegios, Institutos, Colegios Profesionales....

Para utilizarlo, agrega el shortcode: 
[plg_info_alcaide_biblioteca] en cualquier página o entrada.
